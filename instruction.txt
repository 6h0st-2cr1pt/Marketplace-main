
            To Run the Software

Step 1. Open the File in the TEXT EDITOR OR IDE.

Step 2. Create a Virtual Environment ( OPTIONAL )

         Step 2.1: Activate the Virtual Environment using the terminal in TEXT EDITOR OR IDE.
         Step 2.2 Type - [ venv\Scripts\Activate ] in the terminal and press enter.

Step 3 Type [ cd myproject ] in the TEXT EDITOR OR IDE Terminal and press enter.

Step 3: Run the software by typing a command in the Terminal.
        Type - [ python manage.py runserver ]

Step 4: click the provided link in the terminal to open the software.

This will appear in the terminal after you run the command.
